#Thermometer project
Some parts of the code is in Swedish since the project idea was a christmas gift to my dad.

##If you have all components and want to test:
You need to change wifiname, password and API_KEY. You need to set up an API with [ThingSpeak](https://thingspeak.mathworks.com/).

##Componenets used:
[ESP32-C3 Mini](https://www.electrokit.com/esp32-c3-utvecklingskort-with-headers)
[DHT22](https://www.electrokit.com/temp/fuktsensor-rht03)
[DS18B20](https://www.electrokit.com/temperatursensor-vattentat-metall-ds18b20)
[OLED Screen](https://www.electrokit.com/lcd-oled-0.91128x32px-i2c)
